机器人开发学习路线

从零基础到掌握 ROS2 开发的完整学习规划

本学习路线专为机器人开发初学者设计，涵盖从基础工具配置到 ROS2 核心技能的完整学习路径。按照时间规划逐步学习，你将具备独立完成简单机器人项目的能力。

一、学习路线概览

机器人开发需要掌握多项技能，以下是核心学习内容的总体框架：

🎯 学习目标：

掌握机器人开发的核心技能，能够独立完成简单的机器人项目，为参与竞赛和实际开发打下坚实基础。

  • 工具基础：Markdown、Git、GitHub
  • 开发环境：WSL2 + Ubuntu 24.04 + ROS2 Jazzy
  • 编程语言：Python 或 C++ 基础
  • 机器人框架：ROS2 核心概念与实践

二、学习步骤详解

按照以下顺序逐步学习，每个步骤都是后续学习的基础：

1
学习 Markdown 语法
Markdown 是一种轻量级标记语言，广泛用于编写技术文档和学习笔记。学会 Markdown 后，你可以高效地记录学习过程，整理知识体系。
基础工具
2
配置机器人开发环境
安装 WSL2（Windows Subsystem for Linux），并在其中配置 Ubuntu 24.04 系统。然后安装 ROS2 Jazzy 版本，这是机器人开发的核心中间件。
注意：对于 ROS1 项目，需要使用 Ubuntu 20.04 系统。
环境配置
3
安装 Git 并学习版本控制
Git 是最流行的版本控制系统，用于管理代码的修改历史。学会 Git 后，你可以追踪代码变更、回退错误修改、与他人协作开发。
版本控制
4
注册 GitHub 账号
GitHub 是全球最大的代码托管平台，用于存储代码和笔记。注册 GitHub 后，建立自己的学习笔记仓库，将学习成果整理并公开分享。
代码托管
5
掌握 Python 基础
Python 是机器人开发中最常用的编程语言之一。建议掌握 Python 基础语法、数据结构、函数和模块，能够独立完成简单的机器人项目。
编程语言
6
学习 ROS2 核心
ROS2 是机器人开发的核心框架。学习 ROS2 的核心概念（节点 Node、话题 Topic、服务 Service、动作 Action），并通过实践项目加深理解。
机器人框架 🔗 21天学ROS2教程

三、时间安排

以下是针对零基础同学的时间安排建议，可根据个人情况调整。这些是集训的最低要求：

⚠️ 时间安排仅供参考，请根据个人学习进度灵活调整。集训期间必须完成最低要求。
7.13 - 7.15
第一阶段：基础工具与环境
学习 Markdown 语法，配置所需开发环境（WSL2 + Ubuntu + ROS2），安装 Git，注册 GitHub 账号，提交一个自己的仓库，完成学习笔记。
7.16 - 7.18
第二阶段：编程语言基础
学习 Python（或 C++）基础语法，掌握变量、数据类型、循环、函数等核心概念，完成简单的编程练习。
7.19 - 7.25
第三阶段：ROS2 核心学习
学习 ROS2 核心概念，包括节点、话题、服务、动作等。参考鱼香 ROS2 和21天学 ROS2 的教程，完成基础实践项目。
开学前
第四阶段：组队与竞赛准备
完成初步组队，回家之后到开学前继续进行竞赛准备，深入学习机器人相关技术，为竞赛打下坚实基础。

四、推荐资源

以下是学习过程中推荐的教程和资源：

  • 鱼香 ROS2 教程：通俗易懂的 ROS2 入门教程
  • 21天学 ROS2： https://book.guyuehome.com/
  • ROS2 官方文档：权威的 ROS2 技术文档
  • 古月居：机器人技术学习社区
💡 学习建议：

1. 每天保持一定的学习时间，避免临时突击

2. 多动手实践，理论知识需要通过项目巩固

3. 遇到问题及时记录并寻求帮助，不要积累问题

4. 建立学习笔记，整理知识体系

注意事项

  • 以上时间安排是集训的最低要求，请务必按时完成
  • 有基础的同学可以适当加快进度，提前进入下一阶段
  • 遇到困难时，可以向队友或学长学姐寻求帮助
  • 学习过程中要保持耐心，机器人开发需要持续积累
