Linux 入门命令教程

Linux 是开发、人工智能、服务器等领域最常用的操作系统。掌握常见命令后，就可以完成绝大部分日常操作。

一、终端（Terminal）

终端是我们输入命令与 Linux 系统进行交互的窗口。

bash
echo Hello World
输出
Hello World

二、查看当前目录

pwd (Print Working Directory)

用于查看自己当前在系统中所处的绝对路径位置。

bash
pwd
示例输出
/home/qi

💡 上述输出表示当前位于：/home/qi 目录下。

三、查看文件

ls

查看当前目录下有哪些文件和子目录。

bash
ls
示例输出
Desktop Documents Downloads Pictures

常用参数：ls -a

查看所有文件，包含隐藏文件。

bash
ls -a
示例输出
. .. .git .bashrc test.txt

📌 提示： Linux 中以 . 开头的文件或目录都是隐藏文件。

四、切换目录

cd (Change Directory)

1. 进入某个目录

bash
cd Documents

2. 返回上一层目录

bash
cd ..

例如：从 /home/qi/Documents 变成 /home/qi

3. 返回家目录（Home Directory）

bash
cd

或者使用波浪号：

bash
cd ~

4. 进入绝对路径

bash
cd /home/qi/Desktop

五、创建文件和目录

mkdir (Make Directory)

创建新目录：

bash
mkdir project

一次性创建多级嵌套目录：

bash
mkdir -p a/b/c

目录结构如下：

a/ └── b/ └── c/

touch

创建一个空文件：

bash
touch test.txt

同时创建多个文件：

bash
touch a.txt b.txt c.txt

六、查看文件内容

cat

一次性查看整个文件的所有内容（适合小文件）：

bash
cat test.txt

less

分页查看大文件，不会一次性刷屏：

bash
less test.txt

交互快捷键：

  • Space (空格键) ：下一页
  • b ：上一页
  • q ：退出查看

head

默认查看文件的前 10 行：

bash
head test.txt

查看指定的前 20 行：

bash
head -20 test.txt

tail

默认查看文件的最后 10 行：

bash
tail test.txt

查看最后 20 行：

bash
tail -20 test.txt

💥 实用技巧：实时查看日志更新

bash
tail -f log.txt

退出实时查看状态请按：Ctrl + C

七、复制、移动、删除

cp (Copy)

复制文件（将 a.txt 复制一份并命名为 b.txt）：

bash
cp a.txt b.txt

复制整个目录（需要加 -r 参数表示递归）：

bash
cp -r project backup

mv (Move)

移动文件到指定目录：

bash
mv test.txt Documents/

重命名文件：

bash
mv old.txt new.txt

rm (Remove)

删除文件：

bash
rm test.txt

删除目录（需加 -r）：

bash
rm -r project

强制删除目录及其下所有内容（无提示）：

bash
rm -rf project
⚠️ 极其危险的命令：
DANGER
rm -rf /

千万不要在生产环境执行此命令！它会强制删除根目录下的整个文件系统。虽然现代 Linux 普遍做了防护，但仍需保持敬畏。

八、搜索文件

find

在当前目录 . 及其子目录下，查找所有扩展名为 .txt 的文件：

bash
find . -name "*.txt"

仅查找目录类型（Directories）：

bash
find . -type d

仅查找普通文件类型（Files）：

bash
find . -type f

九、搜索内容

grep

在 test.txt 文件中搜索包含指定关键词 hello 的文本行：

bash
grep hello test.txt

十、查看系统信息

查看当前登录的用户名称：

bash
whoami

查看系统当前日期和时间：

bash
date

查看内核版本与完整的系统详细信息：

bash
uname -a

十一、软件安装（Ubuntu）

更新本地包索引软件源：

bash
sudo apt update

升级所有已安装的软件包到最新版本：

bash
sudo apt upgrade

安装指定的软件（例如安装 git）：

bash
sudo apt install git

卸载指定的软件：

bash
sudo apt remove git

十二、网络命令

查看本机的网卡及 IP 地址信息：

bash
ip addr

测试网络连通性：

bash
ping www.baidu.com

💡 提示：Linux 下 ping 会持续进行，可按 Ctrl + C 强行停止。

使用 wget 命令行下载文件：

bash
wget https://example.com/file.zip

或者使用 curl 下载并保存文件：

bash
curl -O https://example.com/file.zip

十三、重定向与管道

1. 输出重定向 >（覆盖模式：将命令结果写入文件，原内容会被清空）：

bash
ls > list.txt

2. 追加重定向 >>（追加模式：在文件末尾新起一行添加内容）：

bash
echo hello >> list.txt

3. 管道 |（将前一个命令的输出结果，作为下一个命令的输入来处理）：

bash
ls | grep txt

🔄 原理解析：ls 罗列出的文件列表 ↓ 传入 grep 进行二次过滤筛选包含 txt 的行。

十四、常用快捷键

快捷键 / 命令功能说明
Ctrl + C强制终止当前正在运行的程序
Ctrl + Z将当前程序挂起后台暂休
Ctrl + D退出当前终端会话 (等同于 exit)
Ctrl + L快速清屏 (保持当前状态)
↑ (方向键上)切换显示上一条执行过的历史命令
↓ (方向键下)切换显示下一条执行过的历史命令
Tab自动补全命令或文件路径 (连按两次列出匹配)
Ctrl + R进入交互式搜索历史命令模式
history直接控制台打印查看所有的历史执行命令

十五、常用命令速查表

常用命令主要功能
pwd查看当前所处的绝对路径
ls列出当前目录下的文件与子目录
cd切换或进入指定目录
mkdir创建新目录
touch创建空文件或更新文件时间戳
cat一并查看输出文件的全部内容
less长文本大文件分页交互式查看
cp复制文件或目录 (-r)
mv移动文件或为文件重命名
rm删除文件或整个目录 (-rf)
find按名称、类型等条件搜索寻找文件
grep强大的文本行关键词检索过滤工具
chmod修改文件或目录的访问权限
tar打包、压缩或解压文件工具
ps快照查看当前系统的活动进程状态
top动态实时查看系统资源占用率及进程排行
kill依据进程 PID 结束终止指定进程
ip addr查看本机网络接口与 IP 配置
ping测试检测与目的主机的网络连通性
wget / curl通过网络链接地址直接下载文件
history显示曾执行过的命令历史列表
clear 或 Ctrl+L清除终端屏幕显示的旧内容
